Learning Java I          

Learn the basics of Java language constructs, object-oriented programming, graphics, event-driven programming. Gain practical experience of creating and modifying Java applications and applets, and embedding Java applets in HTML files. Upon completing the course, students will able to

  • Know the advantages of Java over other programming languages and the significance of Java to the Internet.
  • Become familiar with Java language constructs including decision statements, loop statements, methods, and arrays.
  • Program with classes and objects and use class inheritance.
  • Develop graphical user interface and write event-driven programs.
  • Swing Components
Text: Introduction to Java Programming with JBuilder 3, by Y. Daniel Liang.

 

Learning Java II       

Learn advanced Java features to build comprehensive Java projects. Topics include:

  • Exception Handling
  • Internationalization
  • Multithreading
  • Multimedia
  • Input and Output
  • Networking
Text: Introduction to Java Programming with JBuilder 3, by Y. Daniel Liang.

 

Rapid Java Application Development Using JBuilder

If the "Learning Java Course" is about teaching Java nuts and bolts, this course teaches creating Java reusable building blocks, referred to as JavaBeans, which can be visually manipulated during design time and used as building block for developing software. the JavaBeans component architectures, is the component model for the Java language, a remarkable extension and enhancement to Java, which makes developing software more efficient and more productive. JavaBeans allows you to create reusable software components that can then be assembled using visual application builder tools such as JBuilder, Cafe, PowerJ, Visual Age for Java, and Java Workshop. Upon completing this course, students will be able to

  • Use JavaBeans in rapid application development using JBuilder.
  • Create JavaBeans and install it in JBuilder.
  • Know JavaBeans event model.
  • Become familiar with a large number of representative JavaBeans.
  • Understand bean persistence
  • Use beans in other Bean testing and development tools such as Sun's BeanBox and Visual Basic.
  • Customize beans property editors.
Text: Rapid Java Application Development Using JBuilder 3, by Y. Daniel Liang.

 

Java Database Programming Using JBuilder

JDBC is a Java API that provides Java programmers with a uniform interface for accessing and manipulating a wide range of relational databases. Before JDBC, database clients were typically developed using Microsoft Visual Basic, Borland Delphi, and other development tools provided by database vendors such as Powersoft by Sybase and Oracle Developer 2000 by Oracle. These are excellent rapid application development tools, but the programs developed using these tools run only on certain platforms. With JDBC, you can write database programs to run on any platform that has a Java virtual machine with an appropriate JDBC driver. Upon completing this course, students will be able to:

  • Understand JDBC Architectures.
  • Become familiar with JDBC driver types and know representative JDBC drivers.
  • Use ResultSet to process query results.
  • Get database information from the MetaDatabase class..
  • Process statements (query, update, dynamic SQL, stored procedures).
  • Develop database projects using JBuilder data-aware JavaBeans
Text: Rapid Java Application Development Using JBuilder 3, by Y. Daniel Liang.
